Aufgabe 1 - Responsives Diagramm mit Legende und Titel

Nutze viewBox und relative Breiten für Responsive-Support. Füge einen Titel und eine Legende hinzu, die das Diagramm selbst erklärt.

Startcode (in main_Tage7.js einfügen):

// Tag 7 - Mini-Projekt: Vollständiges, responsives Diagramm

d3.csv("data/lernzeiten.csv", d => ({ tag: d.tag, minuten: +d.minuten }))

  .then(data => {



  const width = 760, height = 460;

  const margin = { top: 64, right: 20, bottom: 52, left: 64 };

  const innerWidth = width - margin.left - margin.right;

  const innerHeight = height - margin.top - margin.bottom;



  const svg = d3.select("#chart")

    .attr("viewBox", `0 0 ${width} ${height}`)

    .style("width", "100%")    // responsiv: passt sich der Container-Breite an

    .style("height", "auto");



  // Titel

  svg.append("text")

    .attr("x", width / 2)

    .attr("y", 32)

    .attr("text-anchor", "middle")

    .attr("font-size", "18px")

    .attr("font-weight", "600")

    .attr("fill", "#1e293b")

    .text("Lernminuten pro Wochentag");



  const g = svg.append("g").attr("transform", `translate(${margin.left},${margin.top})`);



  const x = d3.scaleBand().domain(data.map(d => d.tag)).range([0, innerWidth]).padding(0.25);

  const y = d3.scaleLinear().domain([0, d3.max(data, d => d.minuten)]).nice().range([innerHeight, 0]);



  g.append("g").attr("transform", `translate(0,${innerHeight})`).call(d3.axisBottom(x));

  g.append("g").call(d3.axisLeft(y).ticks(6));



  // Y-Beschriftung

  g.append("text")

    .attr("transform", "rotate(-90)")

    .attr("x", -innerHeight / 2).attr("y", -48)

    .attr("text-anchor", "middle").attr("fill", "#64748b").attr("font-size", "13px")

    .text("Minuten");



  // Balken mit Transition und Tooltip

  const tooltip = d3.select("body").append("div")

    .style("position", "absolute").style("background", "#1e293b")

    .style("color", "#fff").style("padding", "6px 12px")

    .style("border-radius", "6px").style("font-size", "13px")

    .style("pointer-events", "none").style("opacity", 0);



  g.selectAll("rect")

    .data(data)

    .join(enter => enter.append("rect")

      .attr("x", d => x(d.tag)).attr("y", innerHeight)

      .attr("width", x.bandwidth()).attr("height", 0).attr("fill", "#2563eb"))

    .transition().duration(700).delay((d, i) => i * 80).ease(d3.easeCubicOut)

    .attr("y", d => y(d.minuten))

    .attr("height", d => innerHeight - y(d.minuten));



  // Legende

  const legend = svg.append("g").attr("transform", `translate(${margin.left}, ${height - 12})`);

  legend.append("rect").attr("width", 14).attr("height", 14).attr("fill", "#2563eb").attr("rx", 2);

  legend.append("text").attr("x", 20).attr("y", 12).attr("font-size", "12px").attr("fill", "#64748b")

    .text("Lernminuten (Woche KW 15, 2026)");



  // Quellenangabe

  svg.append("text")

    .attr("x", width - margin.right).attr("y", height - 2)

    .attr("text-anchor", "end").attr("font-size", "11px").attr("fill", "#94a3b8")

    .text("Quelle: eigene Messung");

});

Was du dabei lernst

  • Responsives SVG mit viewBox + style("width","100%")
  • Titel, Legende und Quellenangabe als SVG-Text platzieren
  • Alle Techniken aus Tag 16 kombiniert in einem Diagramm
  • Struktur und Lesbarkeit: Kommentierte, wartbare Code-Basis

Aufgabe 2 - Dein eigenes Mini-Projekt

  1. Wähle ein eigenes Thema (Vokabeln/Tag, Gewicht, Schlafzeiten, ...).
  2. Erstelle eine eigene CSV-Datei mit echten oder realitätsnahen Daten.
  3. Baue das Diagramm nach dem heutigen Schema auf und passe Titel, Legende und Farben an.
  4. Lege eine kurze README.md an: Was zeigt das Diagramm, welche Daten, wie starten?
Profi-Tipp: Ein gutes Diagramm erklärt sich selbst. Teste: Würde jemand ohne Erklärung verstehen, was du zeigst? Titel, Achsenbeschriftung und Legende sind Pflicht, kein Schmuck.
