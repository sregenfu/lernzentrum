Tag 1: Datenbindung und Kreise

Heute bindest du Rohdaten an SVG-Elemente und baust daraus ein erstes Kreisdiagramm. Schwerpunkt: selectAll, data, enter, append.

Tag 1 im Detail: D3-Basics und Datenbindung

Das Herzstück von D3.js ist der Data Join: Du verbindest Daten mit DOM-Elementen. D3 kümmert sich darum, welche Elemente neu erstellt, aktualisiert oder entfernt werden müssen.

Aufgabe 1 - Kreise aus einem Array rendern

Ersetze in main.js das Balkendiagramm durch einen Bubble-Chart. Jeder Datenpunkt soll ein Kreis werden - der Wert bestimmt Radius und Farbe.

Startcode (in main.js einfügen):

// Tag 1 - Datenbindung mit Kreisen

const data = [12, 25, 8, 32, 19, 27, 14, 40, 5, 22];



const svg = d3.select("#chart")

  .attr("viewBox", "0 0 760 200");



// Farbskala: kleiner Wert = hellblau, großer Wert = dunkelblau

const color = d3.scaleSequential()

  .domain([0, d3.max(data)])

  .interpolator(d3.interpolateBlues);



// Radiusskala

const r = d3.scaleLinear()

  .domain([0, d3.max(data)])

  .range([8, 50]);



// Kreise aus Daten erstellen

svg.selectAll("circle")

  .data(data)

  .enter()

  .append("circle")

  .attr("cx", (d, i) => 50 + i * 70)   // x-Position je Index

  .attr("cy", 100)                       // y-Position (Mitte)

  .attr("r",  d => r(d))                // Radius aus Datenwert

  .attr("fill", d => color(d));         // Farbe aus Datenwert



// Wert als Text in den Kreis

svg.selectAll("text")

  .data(data)

  .enter()

  .append("text")

  .attr("x", (d, i) => 50 + i * 70)

  .attr("y", 104)

  .attr("text-anchor", "middle")

  .attr("fill", d => d > 20 ? "#fff" : "#1e40af")

  .attr("font-size", "12px")

  .text(d => d);

Was du dabei lernst

  • d3.select("#chart") - ein DOM-Element auswählen
  • selectAll("circle") - eine (noch leere) Auswahl anlegen
  • .data(data) - Array mit der Auswahl verbinden
  • .enter().append("circle") - für jeden Datenpunkt ein neues Element erzeugen
  • .attr("r", d => r(d)) - Attribut aus dem Datenwert berechnen (accessor function)
  • scaleSequential + interpolateBlues - Farbe aus Wert interpolieren

Aufgabe 2 - Selbst ausprobieren

  1. Ändere den Array auf eigene Zahlen (z. B. Vokabeln pro Tag).
  2. Ersetze d3.interpolateBlues durch d3.interpolateOranges.
  3. Ändere die y-Position der Kreise, sodass der Wert die Höhe bestimmt (Andeutung: .attr("cy", d => 200 - r(d))).

Kontrollfrage

Was passiert, wenn im Array mehr Werte sind als Kreise bereits im SVG vorhanden? ? Genau das regelt .enter() - es liefert nur die neuen Datenpunkte, für die noch kein Element existiert.

Profi-Tipp: Öffne die Browser-DevTools (F12) ? Reiter Elements und schau dir an, welche <circle>-Tags D3 erzeugt hat. Das ist die beste Methode, um den Data Join wirklich zu verstehen.
