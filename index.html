Hilfeblock

Wie starte und stoppe ich D3.js?

Starten
  1. Starte den lokalen Server im Ordner english-trainer.
  2. Öffne danach diese Startseite und klicke auf D3.js.
Stoppen
  1. Wechsle in das laufende Terminal.
  2. Drücke Strg + C, um den Server zu beenden.

Wichtig: D3.js funktioniert zuverlässig, wenn der Server im Projekt-Root english-trainer läuft und nicht nur im Unterordner src.
