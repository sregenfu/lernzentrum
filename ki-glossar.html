Themenfilter:

Deep Learning (Tiefes Lernen)

Klick auf eine Karte zum Umdrehen: Vorderseite = Abkürzung/Englisch, Rückseite = Erklärung

ANN
Artificial Neural Network - Künstliches Neuronales Netz
DNN
Deep Neural Network - Tiefes Neuronales Netz mit mehreren Schichten
CNN
Convolutional Neural Network - Spezialisiert auf Bildverarbeitung & Mustererkennung
RNN
Recurrent Neural Network - Für sequentielle Daten wie Text oder Audio
LSTM
Long Short-Term Memory - RNN mit besserem Langzeit-Gedächtnis
GRU
Gated Recurrent Unit - Vereinfachte LSTM Variante
Attention
Mechanismus für das Modell, wichtige Eingabeteile zu fokussieren
Transformer
Moderne Architektur für Sprachverarbeitung - Basis für LLMs
Backprop
Backpropagation - Algorithmus zum Trainieren von neuronalen Netzen
Dropout
Regularisierungstechnik - Deaktiviert zufällig Neuronen beim Training
Batch Norm
Batch Normalization - Normalisiert Eingaben pro Trainings-Batch
Epoch
Eine komplette Durchlauf durch alle Trainingsdaten
Aktiv.Funktion
z.B. ReLU, Sigmoid, Tanh - Nichtlinearität in Neuronen
ReLU
Rectified Linear Unit - Häufige Aktivierungsfunktion (max(0,x))
Optimizer
z.B. Adam, SGD - Algorithmen zur Gewichtsanpassung
Loss Function
Fehlerfunktion - Misst wie weit Vorhersage von echtem Wert entfernt ist
Overfitting
Modell lernt Trainingsdaten zu gut - schwache Generalisierung auf neue Daten
Underfitting
Modell ist zu simpel - schlechte Performance auf Train & Test
Gradient Descent
Optimierungsmethode - Folgt Gradienten bergab zum Minimum
Hyperparameter
Einstellungen VOR dem Training (Lernrate, Batch-Größe, etc.)

Machine Learning (Maschinelles Lernen)

Klick auf eine Karte zum Umdrehen: Vorderseite = Abkürzung/Englisch, Rückseite = Erklärung

ML
Machine Learning - Computerprogramme, die aus Daten lernen
Label
Zielwert/Antwort in Trainings-Daten (z.B. Katze vs. Hund)
Feature
Eingabe-Variable oder Merkmal (z.B. Größe, Farbe)
Supervised
Lernen mit Etiketten/Labels in Trainingsdaten
Unsupervised
Lernen OHNE Labels - findet selbst Muster (z.B. Clustering)
Clustering
Gruppierung ähnlicher Datenpunkte ohne vorgegebene Klassen
Classification
Vorhersage diskreter Klassen (Spam vs. kein Spam)
Regression
Vorhersage kontinuierlicher Werte (z.B. Hauspreise)
Decision Tree
Baum aus Ja/Nein-Entscheidungen zur Klassifizierung
Random Forest
Kombination vieler Decision Trees für bessere Vorhersagen
SVM
Support Vector Machine - Findet beste Trennlinie zwischen Klassen
KNN
K-Nearest Neighbors - Klassifiziert mit ähnlichsten Nachbarn
Accuracy
Anteil korrekter Vorhersagen von alles insgesamt
Precision
Anteil korrekt erkannter positiver Fälle unter allen positiv vorhergesagt
Recall
Anteil korrekt erkannter positiver Fälle unter tatsächlich positiv
F1-Score
Harmonischer Mittelwert aus Precision und Recall
Confusion Matrix
Tabelle mit True Pos./Neg. und False Pos./Neg.
ROC/AUC
Kurve zur Bewertung Klassifizierungs-Performance bei verschiedenen Schwellwerten
Cross-Val.
Trainiere mehrfach mit verschiedenen Train/Test Splits

Data Science (Datenwissenschaft)

Klick auf eine Karte zum Umdrehen: Vorderseite = Abkürzung/Englisch, Rückseite = Erklärung

EDA
Exploratory Data Analysis - Daten visualisieren und verstehen
Outlier
Anomale Datenpunkte, die stark vom Trend abweichen
Distribution
Häufigkeitsverteilung eines Merkmals (Normal, Uniform, etc.)
Correlation
Beziehung zwischen zwei Variablen (wie stark verknüpft)
Missing Values
Fehlende Datenpunkte - erfordern Imputation oder Löschung
Sampling
Auswahl eine Teilmenge aus einer größeren Menge
Normalization
Skaliere Features auf vergleichbaren Bereich (z.B. 0-1)
Standardization
Zentriere auf Null, skaliere mit Standardabweichung
Data Cleaning
Entfernt Fehler, Duplikate, und inconsistente Daten
Feature Eng.
Erstelle neue Features aus bestehenden Daten
Pipeline
Verkettete Schritte: Clean → Transform → Model → Evaluate
Reproducibility
Gleiche Ergebnisse bei wiederholter Ausführung (Random Seed!)
KPI
Key Performance Indicator - Hauptleistungsmessgröße
Baseline
Einfaches Minimum-Modell zum Vergleich
Bias
Systematischer Fehler oder Vereinfachungen im Modell
Variance
Empfindlichkeit gegenüber zufälligen Schwankungen in Trainingsdaten
Dim. Reduction
Reduziere Feature-Anzahl (PCA, t-SNE)
PCA
Principal Component Analysis - Findet Haupt-Varianzen

Python (Programmierung & Datenstrukturen)

Klick auf eine Karte zum Umdrehen: Vorderseite = Abkürzung/Englisch, Rückseite = Erklärung

Mutable
Änderbar nach Erstellung (Listen, Dicts, Sets)
Immutable
Nicht änderbar nach Erstellung (Tuples, Strings, Ints)
Virtualenv
Isolierte Python-Umgebung für Projekt-Dependencies
PIP
Python Package Installer - Installiert externe Pakete
PyPI
Python Package Index - Repository für Python-Pakete
requirements.txt
Datei mit allen benötigten Projekt-Paketen und Versionen
Dict
Schlüssel-Wert Paare {key: value} - mutable
List
Geordnete Sammlung [item1, item2] - mutable
Tuple
Geordnete Sammlung (item1, item2) - immutable
Set
Ungeordnete Sammlung ohne Duplikate - mutable
List Comp.
[x*2 for x in list] - kompakte Listenerstellung
Lambda
Anonyme Funktion: lambda x: x*2
Decorator
@decorator - verändert Funktion ohne Code näher zu ändern
Generator
Lazy Iterator mit yield - speichert nicht alle Werte
Exception
Fehlerbehandlung mit try/except Blöcken
Module
Python-Datei (.py) mit Funktionen und Variablen
NumPy
Numerische Berechnungen mit Arrays und Matrizen
Pandas
Tabellenbearbeitung (DataFrame) - SQL-ähnlich
Matplotlib
Datenvisualisierung - Plots, Grafiken, Charts
Scikit-learn
ML-Algorithmen: Klassifikation, Regression, Clustering

Model Context Protocol (MCP)

Klick auf eine Karte zum Umdrehen: Vorderseite = Abkürzung/Englisch, Rückseite = Erklärung

MCP
Model Context Protocol - Standard für KI-Agentenkompetenzen
Context Window
Anzahl Tokens, die ein Modell verarbeiten kann
Token
Grundeinheit von Text (Wörter, Zeichen, Subwörter)
Tool
Externe Funktion, die ein KI-Agent aufrufen kann
Tool Scope
Zugriff-Berechtigungen eines Werkzeugs definieren
System Prompt
Grundanleitung für Verhalten des KI-Modells
Agent
Autonomes KI System mit Entscheidungslogik
Fallback
Alternative Aktion bei Werkzeug-Ausfall
Handoff
Agent übergibt Kontrolle an einen anderen Agent
Output Schema
Definiert Struktur der Antwort eines Werkzeugs
Input Validation
Überprüfe Werkzeug-Eingaben auf Sicherheit & Korrektheit
Prompt Eng.
Optimierung von Text-Anweisungen für bessere KI-Antworten
RAG
Retrieval-Augmented Generation - Wissen + Generierung kombiniert
Embedding
Numerische Darstellung von Text im mehrdimensionalen Raum
Vector DB
DB für Embeddings mit Ähnlichkeitssuche
Logging
Protokollierung von Agent-Aktivitäten und Fehlern
Rate Limiting
Begrenzung von Anfragen pro Zeit zur Ressourcenschonung
LLM
Large Language Model - Großes Sprachmodell (ChatGPT, Claude)
Fine-Tuning
Anpassung eines vortrainierten Modells an eigene Daten
